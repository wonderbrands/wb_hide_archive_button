The Hide Action Archive Button

The Hide Action Archive Button module is an extension for Odoo designed to hide the "Archive" option from the action menu in the user interface.

Features

01

Global Hide: Allows to globally hide the "Archive" button from the action menu across all models in the system.

02

Selective Hide: Enables users to selectively hide the "Archive" button for specific models. Users can configure a list of models for which the "Archive" button should be hidden.

Screenshots

Step 1: Configuration Settings -> Users & Companies -> Users

Step 2:

  • Hide All Action Archive: When selecting the "Hide All Action Archive" option, the "Applied Models for Hiding Archive" configuration will be hidden, and the hiding of the "Archive" button will be applied to all models.

  • Applied Models for Hiding Archive: Enables users to selectively hide the "Archive" button for specific models. Users can configure a list of models for which the "Archive" button should be hidden.
